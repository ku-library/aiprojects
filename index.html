AI Tools Built for
Academic Libraries
Project Impact Snapshot

Two production-deployed AI tools — LibBee and ResearchBee — that bring intelligent assistance to academic library workflows. LibBee handles patron research queries, builds boolean search strings, classifies intent, and routes requests to the right librarian. ResearchBee guides researchers through journal selection, self-archiving rights, data repository matching, and citation export — with Arabic language support.

Why it stood out

A practical, zero-infrastructure library solution — deployed on GitHub Pages and HuggingFace Spaces at no cost — combining RAG architecture, intent classification, multilingual support, and prompt security into tools that are immediately usable by any academic library worldwide.
